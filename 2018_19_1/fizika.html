  • Az elektrosztatika Gauss törvénye
    • Fluxus: ΔΦ = |E| * Δ|A| * cos(α)
    • Elemi fluxusok összegzése "foltokra": Felületi integrál (Φ = ∫EdA)
    • Gömb felszínére eső fluxus = ∫k * Q / r2dA = k * Q / r2 * 4 πr2 = Q / ε0

      • Ha nem veszi körbe, akkor ugyanannyi erővonal lép be, mint ki, így Φ = 0
      • Zárt felületben ha több töltés is van: Φ = ∑Qi / ε0
    • Töltéseloszlás által létrehozott elektrosztatikus erőtér zárt felületre vett fluxusa megegyezik a zárt felületen lévő össz töltés mennyiség és a vákuum dielektromos állandójának hányadosával
    • Végtelen sík töltéssűrűsége
      • Végtelen síkon ω töltéssűrűség esetén a térerősség E = ω / (2ε0)
    • Gömbfelületen ω töltéssűrűség esetén Φ = gömb∫EdA = E * 4πr2 = ∑Q / ε0
      • Az R sugarú gömbfelület körül r sugarú gömbön: E * 4πr2 = 4πR2ω / ε0 → E = R2ω / r2ε0
    • Végtelen hosszú ω töltéssűrűségű hengerfelületen
      • Alap, fedőlap nem ad járulékot
      • Φ = palást∫EdA = E * 2rπ * L = ∑Q / ε0
      • E = ωR / ε0r
  • Elektromos potenciál

3. EA

  • Térerősség és potenciál kapcsolata
    • ΔU = U(x+Δx, y, z) - U(x, y, z) = dU / dx = -ExΔx
    • Ez igaz y-ra és z-re
    • E = -grad(U)
  • Fémek a statikus villamos (elektromos) térben
    • Bármely fémen belül E = 0
    • Az elektronok a felületen pár atomos rétegben gyülekeznek fel
    • A felület ekvipotenciális felület, E érintőre merőleges
      • Induktív bizonyítás: egyébként addig mozognának az elektronok amíg be nem állna egy ilyen egyensúly
    • Elektrosztatikus térben egy fémüregben E = 0
      • Bizonyítás
      • Gauss tétel: A∫EdA = ∑Q / ε0
      • Mivel Q = 0 → E = 0
      • Másképp: ∫Eds = 0 (konzervatív erőtér)
      • Veszünk egy zárt görbét aminek egyik szakasza (g1) a fémen másik az üregben megy keresztül (g2)
      • g1∫Eds + g2∫Eds = 0 + g2∫Eds = 0
      • ds nem 0, így E = 0
  • Csúcshatás
    • Töltéssűrűség ~ Görbület sugara
      • ω = Q / 4πr2 és U = k Q/r képletekből kijön

4. EA

  • Eltolás vektor
    • Adott egy töltött kondenzátor
    • Ha a fegyverzet közé dielektrikumot teszünk, a szigetelőben dipólusmomentumok jönnek létre
      • Polarizált töltések megjelennek
      • Ahhoz, hogy a fegyverzeten lévő töltés állandó maradjon, a lapokon lévő töltésmennyiség lecsökken
      • ωt2 = ωt1 - ωpolarizált
    • Villamos polarizáció vektora megadja az egységnyi térfogatra vonatkoztatott eredő villamos dipólnyomatékot
    • Értéke épp a felületi töltéssűrűségével egyezik meg
    • ωt = ε0E + ωp → D = ε0E + P (D: eltolásvektor, E: térerősség, P: polarizáció vektora)
    • A∫EdA = Qszabad / ε0 → A∫(D - P) / ε0dA → A∫DdA = ∑Qt

    • Rögzített ponttöltés terének energiája
      • W = ∞R∫ k*Q1Q2 / r2dr = k*Q1Q2 / R

      • W = 1/2 ∑QiUi = 1/2 ∫ρUdV
    • A kondenzátor energiája
      • W = 1/2 ∑QiUi = 1/2 Q*U
      • W = 1/2 (Aω)(E*d) = 1/2 (A*d)(εE)*E → W / V = εE2
